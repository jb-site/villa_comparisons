Villa Data Converter

Convert Google Sheets data to JSON for your villa website

How to use this tool

  1. Open your Google Sheets with villa data
  2. Select all cells (Ctrl+A or Cmd+A)
  3. Copy (Ctrl+C or Cmd+C)
  4. Paste into the input box below
  5. Click "Convert to JSON"
  6. Copy the generated JSON
  7. Paste into data/villas.json in your project

Google Sheets Column Structure

Column Name Required? Description
name ✓ Yes Villa name
location ✓ Yes Location (e.g., "Binibeca, Menorca")
hook ✓ Yes One-line description
villa_location Optional Detailed location description
price ✓ Yes Price (e.g., "£3,300 per family inc flights")
photo ✓ Yes Image URL or relative path
amenities ✓ Yes Comma-separated amenities
link_1_label Optional First booking link label (e.g., "Website")
link_1_url Optional First booking link URL
link_2_label Optional Second booking link label
link_2_url Optional Second booking link URL
Add more link columns as needed: link_3_label, link_3_url, link_4_label, link_4_url, etc.
